## Installation
```
git clone https://github.com/keremketenci/dawn-wave-website.git
cd dawn-wave-website

npm install
# or
pnpm install
# or
yarn install
# or
bun install
```
## Usage
```
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```
